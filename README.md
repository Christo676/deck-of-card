# deck-of-card

## Getting Started

### Prerequisites

```
Java 8
```

### Installing

```
mvn clean install
```

## Endpoints

```
POST localhost:8082/api/v1/poker/decks
PUT localhost:8082/api/v1/poker/decks/1/shuffle
PUT localhost:8082/api/v1/poker/decks/1/cards/draw
```

## Built With

* Spring Boot
* H2 Embedded Database
* Jooq
